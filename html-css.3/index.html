سلام! به سایت من خوش آمدید

به دنیای برنامه‌نویسی و علوم نوین علاقه‌مندم و فعلاً با پایتون کار می‌کنم. جاوا رو هم دارم یاد می‌گیرم و با HTML و CSS تا حدی آشنایی دارم. برنامه‌م برای تابستون اینه که برم سراغ یادگیری JavaScript و Node.js تا بتونم مهارت‌هامو توی توسعه وب بیشتر گسترش بدم. علاقه‌مندی اصلی من به حوزه‌ی Synthetic Biology هست و می‌خوام این زمینه رو به عنوان مسیر تخصصی‌ام ادامه بدم. نکته مهمی که حتماً باید بدونی اینه که دارم زبان ژاپنی یاد می‌گیرم، و این یکی از چیزاییه که واقعاً برام مهمه و با علاقه دنبال می‌کنم. اگه دوست داری باهام در ارتباط باشی، همکاری کنیم یا فقط درباره مسیر یادگیری‌مون گپ بزنیم، خوشحال می‌شم ازت بشنوم :)
